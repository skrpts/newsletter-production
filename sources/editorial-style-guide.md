## Purpose

Defines the editorial standards that all content must follow. The editorial review skill uses this as its primary reference for style compliance checks.

## Formatting Standards

### Headings

- Use sentence case for all headings (capitalise only the first word and proper nouns)
- H1 is reserved for the page title — one per piece
- H2 for major sections, H3 for subsections
- Do not skip heading levels (no H1 to H3 without H2)
- Keep headings under 8 words where possible

### Lists

- Use bullet points for unordered items
- Use numbered lists only when sequence matters
- Start each item with a capital letter
- No full stops on single-phrase items; full stops on complete sentences
- Keep list items parallel in structure

### Punctuation

- Use the Oxford comma (red, white, and blue)
- Use en dashes (--) for ranges, em dashes (---) for parenthetical statements
- Place punctuation inside quotation marks only when it belongs to the quoted material
- Use single quotation marks for quotes within quotes

### Numbers

- Spell out one to nine; use numerals for 10 and above
- Always use numerals with units (5 kg, 3 minutes)
- Use commas in numbers over 999 (1,000 not 1000)
- Spell out percentages in body text (fifteen per cent); use % in tables and headings

### Capitalisation

- Product names: always capitalise as specified by the brand
- Job titles: capitalise only when used as a proper title before a name
- Industry terms: lowercase unless a registered trademark
